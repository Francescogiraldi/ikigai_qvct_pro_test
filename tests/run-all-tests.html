Tests manuels

Pour exécuter les tests automatisés, utilisez les commandes suivantes dans le terminal :

cd /mnt/c/Users/Xx-No/Desktop/Applications/IKIGAI/Duplicate-IKIGAIAPP-04-04-2025-main
npm run test:unit       # Exécute les tests unitaires
npm run test:integration # Exécute les tests d'intégration
npm test                # Exécute tous les tests
